## 🛠 Featured Projects

### 1. Graphene Electron Transport Simulator | C, PETSc, Intel MKL
🔗 [Repository](https://github.com/AlexKKodama/TBG)

**Focus:** *High-Performance Computing & Numerical Linear Algebra*
* Refactored a scientific simulator to evaluate the impact of sparse and dense matrix representations on electron transport computations.
* Migrated the core numerical backend from **Intel MKL** to **PETSc**, improving scalability and flexibility in numerical data structures.
* Reduced execution time by ~79% and memory usage by ~86% compared to the original implementation through PETSc-based optimizations and improved storage strategies.
* Conducted comparative analysis on computational throughput and memory overhead across multiple matrix representations in a low-level C environment.

* Explored core compiler concepts including syntax trees, semantic analysis, type validation, and low-level program representation.
